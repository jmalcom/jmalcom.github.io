UO CIT Minor

The computer science department offers a Computer and Information Technology (CIT) minor, which emphasizes open-source technology and computing skills such as web development, networks, database programming and cloud computing. Read more about the CIT minor at cs.uoregon.edu.

What the CIT Minor to Me

The CIT Minor started with an Introdction to JavaScipt where we learned the basics. Arrays, objects, functions, boolean, Document.querySelector, etc. were a few of the concepts we learned in the course. It was a great introduction computer programming and made me excited to learn more.

Then I took Introduction to Multimedia Programming, where we used Javascript with Google Maps API, canvas, and local storage to create awesome web apps that did cool things

CIT 281 was a great class to have before getting into the upper divison courses as it exposed me to many new concepts like React, Git, and Node to name a few. I'm excited for the next sequence of the CIT Minor.

CIT Courses & Content

  • CIS 111: WebDev w/JavaScipt (ES6)

  • CIT 381: Relational Databases
  • CIT 382: WebDev Frameworks (Meteor & React)
  • CIT 383: Networks & CyberSecurity
